You write JavaScript

JavaScript is a crappy name for the language.

It's not Java, and it's a lot more powerful than the word Script implies.

But you know that.

Writing quality JavaScript is hard...

You have to know what to do...

Wrap your code in closures...

            
(function ($) {
    // Way to go, boss!
}(jQuery));

Namespace your functions...

            
var AwesomeNS = {};

AwesomeNS.rockstar = function () {
    // Good job, rockstar!
}

Decalre your vars up top...

            
AwesomeNS.rockstar = function () {
    var thisIsHow = "we do it";

    // No block scope, here!
}

And what not to do...

Don't use with...

            
bad = "hehe";
with (theNS) {
    // is this theNS.bad or bad.
    // Who knows!
    bad = "oh noes";
}

Avoid eval...

            
var foo = {}, property, value;

foo.bar = "woah";
property = "bar";
// Way to slow down processing, bro!
value = eval("foo." + property);

Always remember your vars...

            
function derp() {
    for (i = 0; i < 10; i++) {
        // You just modified
        // the global 'i'!
    }
}

But you know that, too.
