You are an engineer.

CISQ: Consortium for IT Software Quality says...

  • Reliability

You write JavaScript

Writing quality JavaScript is hard...

You have to know what to do...

Namespace your functions...

            
var AwesomeNS = {};

AwesomeNS.rockstar = function () {
    // Good job, rockstar!
}

Decalre your vars up top...

            
AwesomeNS.rockstar = function () {
    var thisIsHow = "we do it";

    // No block scope, here!
}

And what not to do...

Don't use with...

            
bad = "hehe";
with (theNS) {
    // is this theNS.bad or bad.
    // Who knows!
    bad = "oh noes";
}

Always remember your vars...

            
function derp() {
    for (i = 0; i < 10; i++) {
        // You just modified
        // the global 'i'!
    }
}
